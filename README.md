# Extreme Weather Crop Risk Simulator

## Overview
The Extreme Weather Crop Risk Simulator is a Java-based application designed to estimate potential crop risks caused by extreme weather conditions. The system analyzes environmental factors such as rainfall deviation, temperature variation, wind intensity, and crop resistance levels to calculate a crop risk score.

This project demonstrates how weather variability can affect agricultural productivity and highlights the importance of risk assessment in farming.

## Features
- User-friendly Java Swing GUI
- Simulation of temperature deviation and wind speed
- Calculation of rainfall deviation
- Crop resistance estimation based on crop type
- Risk score computation
- Risk level classification (Low, Moderate, High)
- Advisory recommendations based on risk level

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Java Swing (GUI)
- Random Weather Simulation

## System Architecture
The system consists of the following modules:

Crop Class  
Stores crop information and resistance levels.

Weather Class  
Handles rainfall deviation and weather parameters.

RiskCalculator Class  
Calculates the crop risk score and determines risk levels.

Main Class  
Provides the graphical user interface and integrates all modules.

## How the Simulation Works
1. The user enters crop name and rainfall data.
2. The system simulates weather conditions such as temperature deviation and wind speed.
3. Rainfall deviation is calculated by comparing expected and actual rainfall.
4. Weather parameters and crop resistance levels are analyzed.
5. A crop risk score is computed to estimate the potential impact of extreme weather conditions.

## Example Output
The application generates a risk analysis report including:
- Crop Name
- Temperature Deviation
- Wind Speed Index
- Rainfall Deviation
- Risk Score
- Risk Level
- Advisory Recommendation
